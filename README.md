# README – Installation et utilisation de TeamViewer sous Windows

## Présentation

[TeamViewer](https://www.teamviewer.com/fr/?utm_source=chatgpt.com) est un logiciel de prise en main à distance permettant :

* l’accès à distance à un ordinateur,
* l’assistance informatique,
* le partage d’écran,
* le transfert de fichiers,
* la collaboration à distance.

La version gratuite est disponible pour un usage personnel. ([teamviewer.com][1])

---

# Téléchargement

Téléchargez la dernière version de TeamViewer pour Windows depuis le site officiel :

👉 [Téléchargement TeamViewer Windows](https://www.teamviewer.com/fr/download/windows/?utm_source=chatgpt.com)

Version actuelle disponible : **15.76.6**. ([teamviewer.com][2])

---

# Prérequis

* Système d’exploitation : Windows 10 / Windows 11
* Connexion Internet active
* Droits administrateur recommandés pour l’installation complète

---

# Installation

## 1. Télécharger l’installateur

* Ouvrez le lien officiel de téléchargement.
* Cliquez sur **Télécharger** pour récupérer le fichier `TeamViewer_Setup.exe`. ([teamviewer.com][3])

---

## 2. Lancer l’installation

* Ouvrez le fichier téléchargé.
* Cliquez sur **Exécuter** si Windows affiche un avertissement de sécurité.
* Vérifiez que l’éditeur est bien **TeamViewer Germany GmbH**. ([teamviewer.com][3])

---

## 3. Choisir le type d’installation

Sélectionnez :

* **Installation par défaut** → installation classique recommandée
* **Exécuter uniquement** → utilisation sans installation
* **Accès sans surveillance** → accès permanent à distance

Puis cliquez sur :

```bash
Accepter - suivant
```

---

## 4. Finaliser

* Acceptez le contrat de licence utilisateur (CLUF)
* Attendez la fin de l’installation
* Lancez TeamViewer

---

# Utilisation

## Fournir une assistance

1. Ouvrez TeamViewer
2. Communiquez votre :

   * ID TeamViewer
   * Mot de passe temporaire
3. Le technicien pourra se connecter à votre ordinateur

---

## Se connecter à un ordinateur distant

1. Demandez l’ID TeamViewer du poste distant
2. Saisissez-le dans :

```bash
Contrôler un ordinateur distant
```

3. Cliquez sur :

```bash
Connexion
```

4. Entrez le mot de passe fourni

# Modes disponibles

## TeamViewer Full Client

Permet :

* assistance à distance,
* connexions entrantes et sortantes,
* administration distante. ([teamviewer.com][2])

---

## TeamViewer QuickSupport

Version légère sans installation destinée uniquement à recevoir de l’assistance. ([teamviewer.com][4])

---

## TeamViewer Host

Permet un accès permanent sans validation utilisateur. Idéal pour :

* serveurs,
* maintenance,
* accès à distance 24h/24. ([teamviewer.com][2])

---

# Sécurité

TeamViewer utilise :

* chiffrement AES 256 bits,
* échange de clés RSA 4096 bits,
* connexions sécurisées de bout en bout. ([teamviewer.com][1])
