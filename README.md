# README – Installation et utilisation de TeamViewer sous Windows

## Présentation

[TeamViewer](https://www.teamviewer.com/fr/?utm_source=chatgpt.com) est un logiciel de prise en main à distance permettant :

* l’accès à distance à un ordinateur,
* l’assistance informatique,
* le partage d’écran,
* le transfert de fichiers,
* la collaboration à distance.

La version gratuite est disponible pour un usage personnel. ([teamviewer.com][1])

---

# Téléchargement

Téléchargez la dernière version de TeamViewer pour Windows depuis le site officiel :

👉 [Téléchargement TeamViewer Windows](https://www.teamviewer.com/fr/download/windows/?utm_source=chatgpt.com)

Version actuelle disponible : **15.76.6**. ([teamviewer.com][2])

---

# Prérequis

* Système d’exploitation : Windows 10 / Windows 11
* Connexion Internet active
* Droits administrateur recommandés pour l’installation complète

---

# Installation

## 1. Télécharger l’installateur

* Ouvrez le lien officiel de téléchargement.
* Cliquez sur **Télécharger** pour récupérer le fichier `TeamViewer_Setup.exe`. ([teamviewer.com][3])

---

## 2. Lancer l’installation

* Ouvrez le fichier téléchargé.
* Cliquez sur **Exécuter** si Windows affiche un avertissement de sécurité.
* Vérifiez que l’éditeur est bien **TeamViewer Germany GmbH**. ([teamviewer.com][3])

---

## 3. Choisir le type d’installation

Sélectionnez :

* **Installation par défaut** → installation classique recommandée
* **Exécuter uniquement** → utilisation sans installation
* **Accès sans surveillance** → accès permanent à distance

Puis cliquez sur :

```bash
Accepter - suivant
```

---

## 4. Finaliser

* Acceptez le contrat de licence utilisateur (CLUF)
* Attendez la fin de l’installation
* Lancez TeamViewer

---

# Utilisation

## Fournir une assistance

1. Ouvrez TeamViewer
2. Communiquez votre :

   * ID TeamViewer
   * Mot de passe temporaire
3. Le technicien pourra se connecter à votre ordinateur

---

## Se connecter à un ordinateur distant

1. Demandez l’ID TeamViewer du poste distant
2. Saisissez-le dans :

```bash
Contrôler un ordinateur distant
```

3. Cliquez sur :

```bash
Connexion
```

4. Entrez le mot de passe fourni

---

## TeamViewer Full Client

Permet :

* assistance à distance,

Version légère sans installation destinée uniquement à recevoir de l’assistance. ([teamviewer.com][4])

## TeamViewer Host

Permet un accès permanent sans validation utilisateur. Idéal pour :

* serveurs,

* accès à distance 24h/24. ([teamviewer.com][2])

* chiffrement AES 256 bits,

# README – Installation et utilisation de Google Chrome

## Présentation

[Google Chrome](https://www.google.com/intl/fr/chrome/) est un navigateur web développé par Google.
Il est reconnu pour :

* sa rapidité,
* sa sécurité renforcée,
* sa compatibilité multiplateforme,
* son intégration avec les services Google,
* ses nombreuses extensions disponibles via le Chrome Web Store.

Chrome est compatible avec Windows, macOS, Linux, Android et iOS. ([Google][1])

---

# Téléchargement

Téléchargez la dernière version officielle de Google Chrome :

👉 [Télécharger Google Chrome](https://www.google.com/intl/fr/chrome/?utm_source=chatgpt.com)

---

# Fonctionnalités principales

## Navigation rapide

Google Chrome utilise le moteur Blink ainsi que le moteur JavaScript V8 pour améliorer les performances et la fluidité de navigation. ([Wikipédia][2])

---

## Navigation sécurisée

Chrome intègre Google Safe Browsing afin de détecter :

* les sites malveillants,
* le phishing,
* les téléchargements dangereux,
* les extensions suspectes. ([Aide Google][3])

---

## Synchronisation multi-appareils

En vous connectant avec un compte Google, vous pouvez synchroniser :

* favoris,
* mots de passe,
* historique,
* onglets ouverts,
* paramètres du navigateur.

([Google][1])

---

## Navigation privée

Le mode navigation privée permet de ne pas enregistrer :

* l’historique local,
* les cookies de session,
* les données de formulaires.

([Wikipédia][4])

---

## Extensions Chrome

Le Chrome Web Store permet d’ajouter :

* bloqueurs de publicité,
* outils de développement,
* gestionnaires de mots de passe,
* extensions de productivité.

([Wikipédia][5])

---

# Prérequis

## Systèmes compatibles

* Windows 10 / 11
* macOS
* Linux
* Android
* iPhone / iPad

([Wikipédia][2])

---

# Configuration recommandée

## Activer la protection renforcée

Pour améliorer la sécurité :

1. Ouvrez Chrome
2. Accédez à :

```bash
Paramètres > Confidentialité et sécurité > Sécurité
```

3. Activez :

```bash
Protection renforcée
```

Cette fonctionnalité améliore la détection :

* des téléchargements dangereux,
* des sites frauduleux,
* des extensions malveillantes. ([Aide Google][3])

---

# Fonctionnalités utiles

## Gestionnaire de mots de passe

Chrome peut :

* enregistrer vos mots de passe,
* détecter les mots de passe compromis,
* proposer des connexions automatiques.

([Google][1])

---

## Contrôle de sécurité

Le contrôle de sécurité intégré vérifie :

* les mises à jour,

* les mots de passe compromis.

([Google][1])
